Sudy's HTML Tables cheatsheet

The table below outlines several key elements and commands helpful for building tables in HTML

Elements are shown below in blue and Attributes are in green

Element/Attribute name Brief Description Notes and Tips
<table> Creates a new Table Remember to include all other elements within the table element
<tr> Table Row Enclose data elements (see below) in a row
<td> Table Data Element This is similar to a cell in an excel file
<th> Table Heading The first row, where table column names are specified
colspan colspan = "x"- makes data span x columns Include within td brace e.g. <colspan = "2"> Like Merge Cells in excel
rowspan rowspan = "x"- makes data span x rows Include within td brace e.g. <rowspan = "2"> Like Merge Cells in excel
<thead> Table Head Contain table column headings in this element (within a row)
<tbody> Table Body This should contain all the table's data
<tfoot> Table Foot Used to section off table footer. Can be useful for enclosing totals/subtotals etc.
